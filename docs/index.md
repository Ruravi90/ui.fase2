# Índice de Documentación (.md)

Este directorio (`docs/`) centraliza y clasifica todos los archivos Markdown generados durante el desarrollo y rediseño de **Fase 2 Spa**. Puedes usarlos como referencia para entender el progreso, la visión y los detalles técnicos.

---

## 1. Documentos de Diseño y Planeación (`docs/design/`)

### [design-system.md](./design/design-system.md) (NUEVO)
* **Propósito:** El manual principal de la identidad visual de Fase 2 Spa. Documenta la paleta de colores exacta (con equivalencias Pantone), las familias tipográficas, las reglas de los componentes UI (botones, inputs, sombras) y pautas de animación.
* **Uso:** El documento definitivo ("Single Source of Truth") para mantener el *look and feel* constante en todas las vistas futuras.

### [01-login-redesign.md](./design/01-login-redesign.md)
* **Propósito:** Contiene la propuesta inicial de diseño y arquitectura para la página de login. Aquí definimos la paleta de colores (Serene & Organic), las tipografías (Cormorant Garamond, Outfit) y la estructura visual de "Spa y Consultorio" antes de empezar a programar.
* **Uso:** Útil para diseñadores o desarrolladores que quieran conocer las reglas de estilo globales y las decisiones de UI adoptadas.

---

## 2. Seguimiento y Ejecución (`docs/tasks/`)

### [login-redesign-tasks.md](./tasks/login-redesign-tasks.md)
* **Propósito:** Funciona como un checklist (lista de tareas) que se usó para monitorear el progreso técnico paso a paso durante el rediseño del login.
* **Uso:** Permite auditar qué archivos fueron modificados de forma directa durante la sesión de trabajo.

---

## 3. Reportes y Entregables (`docs/releases/`)

### [01-login-walkthrough.md](./releases/01-login-walkthrough.md)
* **Propósito:** Es el reporte final de entrega del rediseño del login. Describe los cambios implementados, el resultado visual y cómo probar o verificar que todo esté funcionando (como el orbe animado y la validación de contraseñas).
* **Uso:** El punto de entrada principal para revisar el resultado final después de que se aplicaron todos los cambios.
